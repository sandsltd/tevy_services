### Strategic Recommendations

#### Priority 1: Meta Tag Optimization (Week 1-2)

**Title Tag Templates:**
- Service + Location: "Alloy Wheel Repair Devon | Mobile Service | Tevy Services"
- Near Me Focus: "Alloy Wheel Refurbishment Near Me | Same Day Service"
- Commercial Intent: "Professional Alloy Wheel Repairs Exeter | From £45"

**Meta Description Framework:**
- Include: Service + Location + USP + Call to Action
- Example: "Expert alloy wheel repair across Devon. Mobile service, same-day repairs from £45. 5-star rated. Call 01392 XXX XXX or book online."

#### Priority 2: Content Development (Week 2-4)

**New Landing Pages Required:**
1. /alloy-wheel-repair-devon
2. /south-west-alloys-specialist
3. /mobile-tyre-repair-exeter
4. Service area pages for key Devon locations

**Content Optimization Checklist:**
- [ ] Add 300-500 words of unique content per page
- [ ] Include FAQs targeting "near me" searches
- [ ] Add customer testimonials with location mentions
- [ ] Include before/after gallery
- [ ] Add clear pricing information
- [ ] Mobile service coverage map

#### Priority 3: Technical SEO (Week 3-4)

**Schema Markup Implementation:**
```json
{
  "@type": "LocalBusiness",
  "name": "Tevy Services",
  "service": ["Alloy Wheel Repair", "Tyre Repair", "Mobile Service"],
  "areaServed": ["Exeter", "Devon", "South West England"]
}
```

**Internal Linking Strategy:**
- Link service pages to relevant location pages
- Create topic clusters around "alloy wheel services"
- Add contextual links from blog to service pages

#### Priority 4: Local SEO Enhancement (Ongoing)

**Google Business Profile Optimization:**
- Primary category: "Alloy Wheel Repair Service"
- Add all service keywords to business description
- Upload before/after photos weekly
- Respond to all reviews mentioning key services

**Review Generation Campaign:**
- Target: 5 new reviews per month
- Focus on mentions of "alloy wheel repair" and location
- Create automated follow-up for completed jobs
